## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- Gemini API key
- OGame account
- Docker (optional for containerized deployment)

### Installation

```bash
# Clone repository
git clone https://github.com/your-username/ogame-agi.git
cd ogame-agi

# Install dependencies
pip install -r requirements.txt

# Configure credentials
cp .env.example .env
# Edit .env with your credentials
```

### Basic Usage

```bash
# Run strategic AI analysis
python3 test_strategic_ai.py

# Launch autonomous demonstration
python3 simple_autonomous_demo.py

# Full competitive simulation
python3 competitive_agi.py
```
